module selenium-webdriver/io/exec

Spawns a child process. The returned Command may be used to wait for the process result or to send signals to the process.

exec(command, opt_options)

Parameters
commandstring

The executable to spawn.

opt_options{args: (Array<string>|undefined), env: (Object<string, string>|undefined), stdio: (string|Array<?(string|number|$jscomp.scope.stream_module.Stream)>|undefined)}=

The command options.

Returns
Command

The launched command.

Options

Type
{args: (Array<string>|undefined), env: (Object<string, string>|undefined), stdio: (string|Array<?(string|number|$jscomp.scope.stream_module.Stream)>|undefined)}

A hash with configuration options for an executed command.

  • args - Command line arguments.
  • env - Command environment; will inherit from the current process if missing.
  • stdio - IO configuration for the spawned server process. For more information, refer to the documentation of child_process.spawn.
