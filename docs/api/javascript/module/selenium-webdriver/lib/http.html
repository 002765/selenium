module selenium-webdriver/lib/http

Defines an environment agnostic command executor that communicates with a remote end using JSON over HTTP.

Clients should implement the Client interface, which is used by the Executor to send commands to the remote end.

Exported Functions

code

buildPath(path, parameters)

Builds a fully qualified path using the given set of command parameters. Each path segment prefixed with ':' will be replaced by the value of the corresponding parameter. All parameters spliced into the path will be removed from the parameter map.

Parameters
pathstring

The original resource path.

parameters!Object<?, *>

The parameters object to splice into the path.

The modified path.
