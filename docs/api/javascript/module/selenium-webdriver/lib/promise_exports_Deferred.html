class Deferred<T>

Represents a value that will be resolved at some point in the future. This class represents the protected "producer" half of a ManagedPromise - each Deferred has a promise property that may be returned to consumers for registering callbacks, reserving the ability to resolve the deferred to the producer.

If this Deferred is rejected and there are no listeners registered before the next turn of the event loop, the rejection will be passed to the ControlFlow as an unhandled failure.
