UnhandledAlertError.captureStackTrace(error, opt_constructor)

Chrome/v8 specific, adds a stack trace to the error object. The optional constructorOpt parameter allows you to pass in a function value. When collecting the stack trace all frames above the topmost call to this function, including that call, will be left out of the stack trace.
