class Binary

Provides a mechanism to configure and launch Firefox in a subprocess for use with WebDriver.

If created without a path for the Firefox binary to use, this class will attempt to find Firefox when #launch() is called. For OSX and Windows, this class will look for Firefox in the current platform's default installation location (e.g. /Applications/Firefox.app on OSX). For all other platforms, the Firefox executable must be available on your system PATH.

useDevEdition(opt_use)

Specifies whether to use Firefox Developer Edition instead of the normal stable channel. Setting this option has no effect if this instance was created with a path to a specific Firefox binary.

This method has no effect on Unix systems where the Firefox application has the same (default) name regardless of version.

Parameters

Whether to use the developer edition. Defaults to true.
