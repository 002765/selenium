CompoundMutator Mutate Method WebDriver
Mutate a script so by calling all component mutators on it in turn. The original, unmodified script is used to generate a script on the StringBuilder, the "ToString" method of which should be used to get the result. We make use of a StringBuilder rather than a normal String so that we can efficiently chain mutators.
